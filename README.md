## Instalation

- Clone this repo

```bash
git clone https://github.com/jukibenghantachi/virtual-spirit-test.git
cd virtual-spirit-test
```

- Install dependencies

```bash
npm install
```

- Run

```bash
npm run dev
```

## Technology

- ReactJS
- React Redux & Redux Toolkit
- Redux Saga
- React Bootstrap
